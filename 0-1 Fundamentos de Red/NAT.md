El NAT (Network Address Translation) es el proceso que permite que muchos dispositivos usen una sola IP pública para navegar por Internet.

- **Tecnicismo:** Método de remapeo de un espacio de direcciones IP en otro mediante la modificación de la información de las cabeceras IP.
    
- **En jerga (El Portero del Edificio):** Tu router Movistar tiene una sola IP pública (la que ve el mundo). Cuando tu laptop, tu móvil y tu AP quieren salir a Internet, el NAT hace que todos parezcan el Router. Al volver la respuesta, el NAT sabe a quién entregarle cada cosa.
    
- **Por qué existe:** Porque no hay suficientes direcciones IPv4 para que cada dispositivo del mundo tenga una propia.
